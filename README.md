# Desafio DIO - Criando Máquinas Virtuais no Azure

## Sobre o Desafio

Este laboratório teve como objetivo aplicar os conhecimentos adquiridos sobre computação em nuvem por meio da criação e configuração de uma Máquina Virtual no Microsoft Azure.

## Objetivos

* Compreender o processo de criação de uma Máquina Virtual no Azure.
* Conhecer as principais configurações necessárias para o provisionamento de recursos.
* Documentar a experiência utilizando GitHub e Markdown.

## Etapas Realizadas

### 1. Criação da Máquina Virtual

Foi acessado o Portal Azure e iniciado o processo de criação de uma Máquina Virtual.

Durante a configuração foram definidos:

* Nome da máquina virtual;
* Região de hospedagem;
* Sistema operacional;
* Usuário administrador;
* Método de autenticação.

### 2. Configuração dos Recursos

Foram configurados os recursos necessários para o funcionamento da máquina virtual, incluindo:

* Grupo de Recursos;
* Rede Virtual;
* Regras básicas de acesso;
* Configurações de armazenamento.

### 3. Revisão e Implantação

Após a validação das configurações, a implantação foi realizada com sucesso.

### 4. Validação da Máquina Virtual

Foi verificado o status da máquina virtual após a criação, confirmando seu funcionamento correto dentro do ambiente Azure.

## Evidências

As evidências da execução do laboratório encontram-se na pasta **imagens** deste repositório. As capturas de tela documentam as principais etapas do processo, incluindo a criação da máquina virtual, configurações realizadas e a implantação concluída no Microsoft Azure.

## Aprendizados

Durante este laboratório foi possível compreender:

* Conceitos de Infraestrutura como Serviço (IaaS);
* Provisionamento de recursos no Azure;
* Configuração de redes e segurança básicas;
* Gerenciamento de máquinas virtuais em ambiente de nuvem.

## Conclusão

A realização deste desafio permitiu consolidar os conhecimentos sobre criação e gerenciamento de Máquinas Virtuais no Microsoft Azure, proporcionando experiência prática com serviços de computação em nuvem.
